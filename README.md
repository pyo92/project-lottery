## 3. 개발 환경

* Intellij IDEA 2022.3 (Ultimate Edition)
* Java 17
* Gradle 7.6
* Spring Boot 3.0.2

## 4. 기술 스택

* Spring Web
* Spring Data JPA
* Spring Boot DevTools
* Spring Configuration Processor
* Lombok
* Thymeleaf
* Mariadb 10
* Redis
* QueryDSL
* Selenium
* Docker-compose
* AWS EC2
* AWS RDS

## 5. 테스트 환경

* Spock
* Testcontainers
